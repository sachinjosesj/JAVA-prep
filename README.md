# Java Interview Prep – Interactive 14-Day Roadmap

An interactive, single-page Java interview preparation application designed for developers preparing for Java, Spring Boot, Microservices, and backend engineering interviews.

## Features

### 📚 Structured Learning Path

* 14-day interview preparation roadmap
* Daily learning objectives
* Progressive difficulty levels
* Interview-focused content

### 🎯 Interview Question Bank

* Frequently asked Java interview questions
* Difficulty classification (Easy, Medium, Hard)
* Detailed answers and explanations
* Real-world examples

### 💻 Code Examples

* Syntax-highlighted Java code snippets
* Spring Boot examples
* Collections, Streams, Concurrency examples
* Best practices and interview tips

### 🧠 Core Topics Covered

#### Java Fundamentals

* OOP Concepts
* Collections Framework
* Exception Handling
* Generics
* String Internals

#### Modern Java

* Streams API
* Functional Interfaces
* Optional
* Lambda Expressions
* Records & Sealed Classes

#### Advanced Java

* Multithreading
* Concurrency
* JVM Internals
* Memory Management
* Garbage Collection

#### Spring Framework

* IoC & Dependency Injection
* Bean Lifecycle
* Spring AOP
* Spring Boot
* Microservices Concepts

### 🎨 User Experience

* Dark theme inspired by GitHub
* Responsive layout
* Collapsible topic sections
* Interactive Q&A cards
* 4K monitor optimized
* Fast and lightweight (No framework required)

## Getting Started

### Clone Repository

```bash
git clone https://github.com/your-username/java-interview-prep.git
cd java-interview-prep
```

### Run Locally

Simply open the HTML file in your browser:

```bash
open java-interview-prep.html
```

or

```bash
double-click java-interview-prep.html
```

No installation required.
